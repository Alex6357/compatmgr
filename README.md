# compatmgr
## FreeBSD Linux兼容层管理器

已可用的功能： 
- [x] 环境检查
- [x] 安装兼容层
- [x] 基于脚本的i18n支持

计划添加的功能：
- [ ] 移除兼容层

已支持的发行版：  
- [x] Debian 12 (bookworm)
- [x] Debian 11 (bullseye)
- [x] Ubuntu 22.04 (jammy)
- [x] Ubuntu 20.04 (focal)
